# **App Name**: Mitho sambandha

## Core Features:

- Homepage Portal: A public-facing homepage that serves as a high-level overview of the platform. It will feature a prominent hero image, a clear "Join Now" call to action, and a curated, scrollable feed of featured profiles to showcase the quality of our members.
- Profile Submission: A multi-step, user-friendly form where new users can submit their details. This form will collect all required information, including name, age, location, a minimum of three high-quality photos, a detailed bio, and partner preferences. All submissions will be pending review by the admin team.
- Profile View: A read-only profile page that displays a member's full details. This page is accessible from the homepage and is designed to provide a comprehensive look at a potential match without allowing any direct user-to-user interaction before a match is approved.
- Admin Dashboard: A private, secure dashboard for the admin team. The dashboard will provide a clear overview of the system, including a queue of new profile submissions, a list of pending updates, and a search tool.
- Profile Approval: A dedicated admin tool for reviewing new and updated profiles. Admins will verify the photos, content, and details before a profile is approved and made public on the homepage.
- Matchmaking Tool: A powerful admin-only search and filtering tool. This tool will allow admins to search the database of approved profiles to find potential matches based on user-submitted preferences, demographics, and other key criteria.
- AI Match Suggestions: This is an advanced feature for the admin dashboard. The AI will analyze approved profiles and partner preferences to suggest potential matches to the admin team, streamlining the matchmaking process and improving efficiency.

## Style Guidelines:

- Primary (#A34A2B): A muted reddish-brown that evokes a sense of heritage, warmth, and tradition. Used for main buttons, headlines, and key branding elements.
- Background (#F2EADF): A very light tan, almost off-white. This soft, inviting background allows the profile images and text to stand out, reducing eye strain and providing a comfortable user experience.
- Accent (#6A994E): A dark olive green. This color will be used sparingly for calls to action, important notifications, and to add a subtle visual contrast that guides the user's eye.
- Headlines & Titles: 'Alegreya', a serif font that provides a feeling of timeless elegance and warmth.
- Body Text & UI: 'PT Sans', a sans-serif font chosen for its excellent readability and clean, modern aesthetic.
- Grid-Based Layout: A clean, organized grid will be used to present profiles and information, making it easy to scan and digest.
- Simple Icons: Consistent, simple icons will be used for profile details like age, location, and interests. These will be visually clear and enhance the user interface without being distracting.
- User-Centric Design: The interface will prioritize a simple and intuitive experience for the user while providing powerful tools for the admin team.
